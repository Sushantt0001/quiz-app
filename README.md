# 🧠 Interactive Quiz App with Timer

A modern and interactive Quiz Web App built using **HTML, CSS, and JavaScript**, featuring a **5-second timer per question**, category selection, and score tracking.

---

## 🚀 Features

- 🎯 Multiple Categories (General, Tech, History, Math)
- 📊 Difficulty Levels (Easy, Medium, Hard)
- ⏱️ 5-Second Timer per Question
- ❌ Auto-submit if time runs out
- ✅ Instant answer feedback (Correct/Wrong)
- 📈 Final Score Display
- 🔁 Restart Quiz option
- 🎨 Clean and responsive UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## ▶️ How to Run Locally

1. Download or clone the repository
2. Open `index.html` in your browser
3. Click "Start Quiz"
4. Select category and difficulty
5. Answer questions within time
